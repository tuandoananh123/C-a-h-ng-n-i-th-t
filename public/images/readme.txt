Add product images to this folder with the following naming convention:

1. hero-1.jpg, hero-2.jpg, hero-3.jpg - For homepage carousel
2. category-kitchen.jpg, category-bathroom.jpg, category-bedroom.jpg, category-livingroom.jpg, category-electronics.jpg, category-furniture.jpg - For category cards
3. product-1.jpg, product-2.jpg, etc. - For product images

You can use any high-quality images of household goods.
